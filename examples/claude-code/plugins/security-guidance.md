# Security Guidance Plugin

## One-line Summary

A PreToolUse hook plugin that monitors 9 security patterns (command injection, XSS, eval usage, dangerous HTML, pickle deserialization, os.system calls, etc.) and warns users when potentially insecure code is being written.
