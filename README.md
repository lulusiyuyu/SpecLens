## What is SpecLens?

SpecLens is a **Skills / Prompt template system** (pure Markdown files) that can be loaded into any AI Coding Agent to:

1. **Learn & Understand** — Decompose an existing codebase into layered Specs for rapid human comprehension
2. **Develop & Maintain** — Use Specs as the top-level unit of maintenance; modify Specs to drive Agent code changes

## Quick Start

1. Download or clone this repo
2. Feed a Skill file (e.g. `skills/analyze/SKILL.md`) to your AI Agent
3. Point it at a codebase and watch it generate Specs
